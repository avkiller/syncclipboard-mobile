
## 🎯 设计原则

**重要**: 如果一个功能模块只有一个文件，则不创建子目录，直接放在父目录下。

**示例**:

- ✅ `utils/Logger.ts` - 单文件模块
- ✅ `api/AuthService.ts` - 单文件模块
- ✅ `api/clients/APIClient.ts` - 多文件模块，需要子目录
